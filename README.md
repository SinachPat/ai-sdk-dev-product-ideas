# AI SDK Product Ideas

Interactive research report mapping 10 high-demand product opportunities to Vercel AI SDK capabilities.

## Deploy in 30 seconds

### Option 1 — Vercel CLI (recommended)
```bash
npm i -g vercel     # install Vercel CLI if you don't have it
vercel --prod       # deploy from this folder
```

### Option 2 — Vercel Dashboard
1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → Import your repo
3. Click Deploy — no build config needed

### Option 3 — Ask Claude Code
Open this folder in Claude Code and say:
> "Deploy this to Vercel"
